# Los Cypherpunks Escriben Código

Perfiles de los protagonistas del movimiento Cypherpunk, organizados por facción filosófica.

Basado en la serie documental de 4 partes **"Cypherpunks Write Code"** de [Reason TV](https://www.youtube.com/watch?v=9vM0oIEhMag).

## 🔶 Las 4 facciones

| Facción | Filosofía | Miembros clave |
|---------|-----------|----------------|
| **Cofundadores** | "Los Cypherpunks escriben código" — la privacidad solo se defiende construyendo | Timothy C. May, Eric Hughes, John Gilmore |
| **Criptoanarquistas** | La criptografía destruirá el monopolio del Estado sobre el dinero y la identidad | David Friedman, Adam Back, Robert Hettinga, Jim McCoy, Zooko Wilcox |
| **Hayekianos de Alta Tecnología** | Transformar la sociedad desde dentro con herramientas de coordinación descentralizada | Phil Salin, Mark S. Miller, Chip Morningstar, E. Dean Tribble, Gayle Pergamit |
| **Criptógrafos** | Los armeros — inventaron y distribuyeron las armas matemáticas | Whitfield Diffie, Phil Zimmermann |
